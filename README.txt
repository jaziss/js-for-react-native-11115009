Task 1: processArray function
- Takes an array of numbers.
- Returns a new array where even numbers are squared and odd numbers are tripled.

Task 2: formatArrayStrings function
- Takes two arrays:
    1. An array of strings.
    2. An array of numbers processed by processArray function.
- Modifies each string based on its corresponding number:
    - Capitalize the entire string if the number is even.
    - Convert the string to lowercase if the number is odd.

Task 3: createUserProfiles function (userInfo.js)
- Takes an array of names and the array of modified names from Task 2.
- Returns an array of objects, each containing:
    - originalName
    - modifiedName
    - id (auto-incremented starting from 1).
